## Como Executar
1. Intalar o [Docker](https://www.docker.com/)
2. Abrir o terminal na raiz do projeto
3. Rodar o comando `docker-compose up --build`
4. Aguardar a criação do container
5. Mudar o nome do arquivo `.env.exemple` para `.env`
